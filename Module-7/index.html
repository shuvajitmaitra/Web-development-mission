Measuring Units

  1. px: It is a fixed unit. It can not change
  2. em: It is changing unit. It is change across the parent div font size
  3. rem:It is also a changing unit. It is change according to the root element font size
  4. percentage: it is calculate according to the parent div. if it's parent div font size is 10 px and child div font size is 200% then child div font size will be 20px
  5. vh: It is calculate according height of the visible part of the display. It cover full visible part of the display
  6. vw: It is also same as vh. But it calculate from the width screen. It cover full of the screen width
  7. Vmin: It is count the most smallest side for the device. Whatever it is hight or width doesn't matter
  8. Vmax: It is count the most biggest side for the device. Whatever it is hight or width doesn't matter
